# Gaula Fiskestatus

En statisk nettside klar for GitHub Pages.

## Publisering

1. Opprett en ny repository på GitHub.
2. Last opp alle filene fra denne mappen til roten av repositoriet.
3. Åpne **Settings → Pages**.
4. Velg **Deploy from a branch**, `main` og `/ (root)`.
5. Trykk **Save**.

Nettsiden inneholder norsk/engelsk språkvalg. Tallene er foreløpig eksempeldata og er tydelig merket som dette.
